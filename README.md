## Description

## Intall node packages

```bash
$ npm install
```

## Copy .env.example to .env file and fill it
```bash  
cp .env.example .env
```

## Compile and run the project

```bash
# development
$ npm run start

# or watch mode
$ npm run start:dev
```

## Requirements
- Node.js = 20.x
- npm = 10.x
- MySQL = 8.x

## API Documentation (Swagger)
The Swagger documentation for the API can be found at http://localhost:3001/api
